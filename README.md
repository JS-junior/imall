## Features

- Products and reviews
- Add to Cart
- Full Stripe payment integration
- Refund
- Search
- JWT authentication

## Installation

Installation

```bash
git clone https://github.com/JS-junior/ecommerce
cd ecommerce
npm i
cd shopping
pip install -requirements.txt
```

## run locally

```bash
expo start
python3 manage.py runserver
```
